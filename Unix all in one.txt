7 a) Write a shell script which displays a list of all the files in the current directory to which you have read, write and execute permissions.


clear

for i in *
do
 if [ -r $i -a -w $i -a -x $i ]
 then
  echo File $i has read, write and execute permission
 fi
done

clear

9a)A shell script receives even number of filenames as arguments. Suppose four files are supplied as arguments then the first file should get copied into second, third file into fourth and so on. If odd number of filenames is supplied then no copying should take place and an error message should be displayed

clear

for i in $*
do
 if [ `expr $# % 2` -eq 0 ]
 then
  cp $1 $2
  shift 2
 else
  echo ERROR - ODD NUMBER OF ARGUMENTS ARE PASSED
 fi
done

10 c)Write script average which computes the average value of the numeric values given in the parameters and writes the result to standard output.

clear

sum=0
count=$#

if [ $count -lt 1 ]
then
 echo NO ARGUMENTS ENTERED
 exit
fi

for i in $*
do
 sum=`expr $sum + $1`
 shift
done

average=$(echo "scale=2; $sum / $count" | bc)

echo Average of given $count numbers is $average
